# ABPSHOP微信小程序多商户电子商城

#### Description
AbpShop 是基于.Net core ABP框架基础开发的以会员为中心的电商系统，小程序商城带生活圈、商户、积分、优惠券、预约等功能，更是一套方便二次开发的商城框架（后台封装了独有快速创建表单功能，无需写表单页面、快速创建数据搜索和数据列表页、导出表格、系统权限配置控制每一个控制器方法、系统参数配置、数据字典、组合数据等）
